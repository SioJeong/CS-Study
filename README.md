## 스터디 규칙
1. 1주일에 교재에서 주제 선정하여 개별 학습 후, 내용 정리
2. 학습한 부분에서 면접 질문 선정과 답변 정리
3. git에 정리

## 파일 업로드 규칙
1. 큰 주제는 폴더로 생성
2. 작은 주제는 폴더 안 Markdowm 파일로 생성
3. 하단에 면접 질문 생성 및 답변은 토글로 바로 보이지 않도록 작성

## 커밋 메세지 규칙

ex) [Network] 2.1 네트워크의 기초
